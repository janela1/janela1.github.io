First 1/4 of the Semester

To start off the course, we first learned about the history of the internet and how it actually works. After setting the stage that way, Dr. Joseph-Nichols started us off with the bare basics of HTML (not CSS yet). We learned the components of the language first, like learning common elements, semantic tagging, and knowing what to include in a code layout. After we could successfully make things appear in a browser (which requires learning how to format files correctly), we ended this portion of the semester by learning the more intricate features that HTML offers. One such example can be seen at right, when we learned how to create a form with multiple types of user input, such as text boxes, radio buttons, and drop-down selections.

Second 1/4 of the Semester

After getting comfortable with the HTML basics, Dr. Joseph-Nicholas then introduced CSS. With CSS styling, we were able to transform our ugly default HTML browser setttings into unique, customizable pages! We learned about how to implement a style sheet, how to structure the code, and about many useful and important CSS style attributes! For example, if you look at the example at right, we learned how to customize a table, the bare bones of which is coded in HTML. When incorporating CSS, you can change the fonts, sizing and alignment of cells, the page background, borders, and a million other things you probably didn't know existed!

Third 1/4 of the Semester

Now that we had our grip on HTML and CSS together, it was time to learn how to actually effectively make a page layout. The major way we accomplished this was by learning through the Box Model and new HTML5 attributes. Because you have to understand how elements are contained and have a good mental map of the page to know how something might be affected, this is where the code becomes a little harder to implement. An example of a layout that incorporates CSS styling, images, and form inputs can be seen at right. A quick hint for this portion of the class: divs are key, and do the majority (if not all) of your styling in your CSS sheet instead of the HTML document!

Final 1/4 of the Semester

This brings us to the last portion of the semester. Familiarity with creating layouts allowed us to learn about more current code for web developers. For example, Bootstrap is a popular source for web developers that allows you to utilize many preset attributes common in the modern webpage without having to do a bunch of styling in CSS. We learned how to properly link our HTML documents to the Bootstrap server, how to customize preset styling, and how to create responsive layouts using their grid system.
